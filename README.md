## ✨ Key Features & Technical Highlights

This project pushes the boundaries of pure HTML, CSS, and JavaScript, implementing features typically reserved for heavy frameworks:

* **3D Tilt & Holographic Cards:** Utilizes complex CSS `transform-style: preserve-3d` and JavaScript mouse-tracking mathematics to create a dynamic, physical tilt effect on the service cards.
* **Glassmorphism UI:** Implements modern frosted-glass aesthetics using CSS `backdrop-filter` for a premium, high-tech feel.
* **Custom Glowing Cursor:** Replaces the default system cursor with a custom DOM element that tracks mouse movement and interacts dynamically with clickable elements.
* **Dynamic Typing Effect:** A custom Vanilla JavaScript function that loops through an array of strings, simulating a realistic typewriter effect with backspacing and blinking cursors.
* **Intersection Observer Animations:** Uses the native browser Intersection Observer API to trigger smooth, hardware-accelerated scroll animations as elements enter the viewport.
* **Fully Responsive Design:** A mobile-first approach ensuring the layout, typography, and custom mobile hamburger navigation scale perfectly across all device sizes.
* **Interactive Scroll Progress:** A dynamic top-bar indicator that calculates the user's scroll depth in real-time.

## 🛠️ Tech Stack

* **HTML5:** Semantic markup and structured content.
* **CSS3:** Advanced animations (`@keyframes`), CSS variables for theme management, Flexbox/Grid for layout, and media queries for responsiveness.
* **Vanilla JavaScript (ES6+):** Pure DOM manipulation, event listeners, and performance-optimized scroll tracking. No external libraries (e.g., jQuery, React, or GSAP) were used.
